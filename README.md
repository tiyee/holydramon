# holydramon

A minimalist Go web framework



## 动机

主要是做一款简单，顺手，可控的go web framework.大约有以下几个痛点：

- 文件少，可控。
- 路由不需要支持正则
- 中间件可以指定范围



## 使用方法

1. `git clone git@github.com:tiyee/holydramon.git`
2. cd holydramon
3. sh ./build.sh
4. sh ./run.sh -c conf/application.toml
